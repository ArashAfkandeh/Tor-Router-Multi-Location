# Tor-Router-Multi-Location
برنامه Tor Router یک ابزار قدرتمند مدیریت Tor است که یک سرور واحد را به چندین لوکیشن مجازی با IP و کشور خروجی متفاوت و قابل تنظیم تبدیل می‌کند. این ابزار با پشتیبانی از چندین تونل مستقل، چرخش خودکار مسیرها، مانیتورینگ زنده، CLI، SQLite و API منعطف، راهکاری ایده‌آل برای مدیریت حرفه‌ای مسیرهای Tor و حفظ حریم خصوصی است. 🇺🇳
